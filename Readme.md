Arduino Library for TZM902DP(ZETA Module)

-------------------------

## TZM902DP(ZETA Module) とは

TZM902DP(ZETA Module)とは、TOPPAN株式会社 から発売の ZETA 通信モジュールです。<br>
このモジュールを搭載した Spresense 向けの Add-onボードも発売され、Arduino IDEでも使用が可能です。

詳細は、下記商品サイトをご覧ください。

![](http://www.toptdc.com/global-image/units/img/16553-1-20211012174158_b61654a569410b.PNG)

TZM902DP : http://www.toptdc.com/product/zeta/

TOPPAN株式会社：https://www.toppan.com/ja/

### API

#### begin

  引数： なし

  戻り値： なし
  
  説明：
	ライブラリを初期化を行います。。

#### end

  引数： なし

  戻り値： なし
  
  説明：
        ライブラリを破棄します。

#### connect

  引数： なし
  
  戻り値： int wait_time : 登録が完了までに待つ時間です。
  
  説明：
        APへこのモジュールを登録し通信の準備を行います。

#### activate

  引数： なし
  
  戻り値： bool : 成否
  
  説明：
        通信を開始するためのコマンドを発行します。

#### read

  引数：
          char* store  : 受信データを格納するバッファアドレス
          int size     : 受信データを格納するバッファサイズ
  
  戻り値： int : 読み出しデータのサイズ
  
  説明：
        受信したデータを読み出します。データは引数に指定したバッファに格納されます。

#### send

  引数：
          char* var  : 送信するデータのの先頭アドレス
          int size   : 送信するデータのサイズ
  
  戻り値： bool : 成否
  
  説明：
        データを送信します。成功するまで3回送信を繰り返します。

#### send_no_retry

  引数：
          char* var  : 送信するデータのの先頭アドレス
          int size   : 送信するデータのサイズ
  
  戻り値： bool : 成否
  
  説明：
        データを送信します。送信を失敗しても繰り返しません。

#### send_raw

  引数：
          char* var  : 送信するパケットのの先頭アドレス
          int size   : 送信するパケットのサイズ
  
  戻り値： bool : 成否
  
  説明：
        パケットを送信します。内部でパケットヘッダーを生成しないので自体を指定してください。
        送信を失敗しても繰り返しません。

#### send_raw

  引数：
          char* var  : 送信するパケットのの先頭アドレス
          int size   : 送信するパケットのサイズ
  
  戻り値： bool : 成否
  
  説明：
        パケットを送信します。内部でパケットヘッダーを生成しないので自体を指定してください。
        送信を失敗しても繰り返しません。

#### downlink_check

  引数： なし

  
  戻り値： int : 応答またはダウンリンクパケットのタイプ
  
  説明：
        ZETAモジュールから応答またはダウンリンクを取得します。

#### get_downlink

  引数：
          char* data  : ダウンリンクデータを格納するバッファアドレス
          int size    : ダウンリンクデータを格納するバッファサイズ
  
  戻り値： int : ダウンリンクデータのサイズ
  
  説明：
        ダウンリンクしたデータを読み出します。データは引数に指定したバッファに格納されます。
        ダウンリンクデータのサイズが戻ります。
